[프로모션 등록 DB 테이블]

* 월별 프로모션을 타겟 고객에 맞춰 프로모션을 정리하는 데이터 베이스이다.  등록, 수정, 삭제가 가능해야 한다.
** 테이블 구조 **
프로모션 ID = 일련번호
프로모션 코드 = 레시피코드 (자동생성)
프로모션 월 = 프로모션이 적용되는 월
타겟 = 고객그룹 및 날짜 매칭
제목 = 제목
인사말 = 간단한 서두 인사말 (사이즈 가변 텍스트필드)
프로모션 내용 = 회사별 추천 제품 및 요금, 사은금 등 기술
매듭말 = 마지막 안내 멘트말
연락처 = 연락처, 카톡 주소 등 (텍스트 필드)

---

[고객 DB]

* 고객 연락처와 보유 상품을 기록한다

** 테이블 구조 **
고객 ID : 일련번호
고객 그룹 : 고객의 성격
고객명 : 고객이름
휴대폰번호 : 휴대폰번호
현재상품 : 현재상품
설치일자 : 설치일자
의무사용기간 : 3년? 5년? 6년 등

---

[문자발송 데이터]

** 테이블 구조 **
문자 ID : 일련번호
프로모션  ID : 프로모션 테이블 참조
고객 ID : 고객 정보 테이블 참조
문자제목 : 문자제목
발송일자 : 발송일자
