# 프로모션 커스텀 태그 참조 가이드

## 개요
이 문서는 렌탈톡톡 프로모션 시스템에서 사용되는 모든 커스텀 태그들을 정리한 참조 가이드입니다. 프로모션 내용 작성 시 이러한 태그들을 활용하여 더욱 시각적으로 매력적이고 구조화된 콘텐츠를 만들 수 있습니다.

## 📋 커스텀 태그 목록

### 1. 콘텐츠 구분 태그

#### `===` - 콘텐츠 그룹 분리
- **용도**: 프로모션 내용을 여러 그룹으로 분리
- **효과**: 각 그룹이 별도의 카드 형태로 표시되며, 그룹 간에 구분선이 자동으로 추가됨
- **사용 예시**:
  ```
  첫 번째 그룹 내용
  ===
  두 번째 그룹 내용
  ===
  세 번째 그룹 내용
  ```

### 2. 시각적 구분선 태그

#### `---` - 두꺼운 가로선
- **용도**: 콘텐츠 내에서 강한 구분이 필요한 경우
- **효과**: 그라데이션이 적용된 두꺼운 가로선으로 표시
- **스타일**: 
  - 높이: 2px
  - 그라데이션: 투명 → 회색 → 투명
  - 상하 여백: 2.5rem
- **사용 예시**:
  ```
  위쪽 내용
  ---
  아래쪽 내용
  ```

#### `...` - 점선 구분선
- **용도**: 콘텐츠 내에서 약한 구분이 필요한 경우
- **효과**: 점선 형태의 가로선으로 표시
- **스타일**:
  - 높이: 1px
  - 점선 패턴: 2px 점 + 4px 간격
  - 상하 여백: 0.75rem
- **사용 예시**:
  ```
  위쪽 내용
  ...
  아래쪽 내용
  ```

### 3. 텍스트 강조 태그

#### `<<텍스트>>` - 파란색 소제목 타이틀
- **용도**: 중요한 소제목이나 강조하고 싶은 텍스트
- **효과**: 파란색 배경의 둥근 모서리 박스로 표시
- **스타일**:
  - 배경: 파란색 그라데이션 (#dbeafe → #bfdbfe)
  - 테두리: 파란색 (#93c5fd)
  - 글자색: 진한 파란색 (#1e40af)
  - 폰트 크기: 1.125em (18px)
  - 폰트 굵기: 600
  - 패딩: 4px 12px
  - 둥근 모서리: 8px
- **사용 예시**:
  ```
  <<특별 혜택>>
  이번 달만의 특별한 할인 혜택을 제공합니다.
  ```

#### `[[텍스트]]` - 회색 그룹박스
- **용도**: 부가 정보나 참고사항을 표시
- **효과**: 회색 배경의 둥근 모서리 박스로 표시
- **스타일**:
  - 배경: 연한 회색 (#f3f4f6)
  - 테두리: 회색 (#d1d5db)
  - 글자색: 진한 회색 (#374151)
  - 폰트 굵기: 500
  - 패딩: 4px 8px
  - 둥근 모서리: 6px
- **사용 예시**:
  ```
  [[참고사항]]
  이 혜택은 신규 고객에 한해 적용됩니다.
  ```

#### `!!텍스트!!` - 연한 분홍색 배경
- **용도**: 주의사항이나 중요한 안내사항
- **효과**: 연한 분홍색 배경으로 텍스트 강조
- **스타일**:
  - 배경: 연한 분홍색 (#fce7f3)
  - 글자색: 기본 텍스트 색상 유지
- **사용 예시**:
  ```
  !!중요!!
  신청 마감일은 12월 31일까지입니다.
  ```

#### `""텍스트""` - 연한 파란색 배경
- **용도**: 부가 설명이나 추가 정보
- **효과**: 연한 파란색 배경으로 텍스트 강조
- **스타일**:
  - 배경: 연한 파란색 (#e0f2fe)
  - 글자색: 기본 텍스트 색상 유지
- **사용 예시**:
  ```
  ""추가 혜택""
  카드 결제 시 추가 5% 할인
  ```

### 4. 자동 링크 처리

#### URL 자동 링크
- **용도**: http:// 또는 https://로 시작하는 URL을 자동으로 클릭 가능한 링크로 변환
- **효과**: 
  - 굵은 글씨체
  - 밑줄 표시
  - 호버 시 색상 변경
- **스타일**:
  - 기본 색상: 남색 (#1e3a8a)
  - 호버 색상: 진한 남색 (#1e40af)
  - 폰트 굵기: bold
  - 텍스트 장식: underline
- **사용 예시**:
  ```
  자세한 내용은 https://example.com 에서 확인하세요.
  ```

- **연한 파란색**: 진한 파란색 (#0c4a6e)
- **링크**: 밝은 파란색 (#3b82f6)

## 📱 반응형 디자인

모든 커스텀 태그는 모바일 환경에서도 최적화되어 표시됩니다:

### 모바일 최적화
- **패딩 조정**: 더 작은 화면에 맞게 패딩 축소
- **여백 조정**: 그룹 간 여백 최적화
- **폰트 크기**: 가독성을 위한 적절한 크기 유지

## 💡 사용 팁

### 1. 구조화된 콘텐츠 작성
```
<<프로모션 소개>>
안녕하세요! 이번 달 특별한 혜택을 준비했습니다.

===

<<주요 혜택>>
- 20% 할인
- 무료 배송
- 사은품 증정

!!중요!! 신청 마감일은 12월 31일까지입니다.

===

<<신청 방법>>
1. 홈페이지 방문
2. 상품 선택
3. 결제 완료

[[참고사항]]
이 혜택은 신규 고객에 한해 적용됩니다.

===

<<문의처>>
자세한 내용은 https://example.com 에서 확인하세요.
```

### 2. 시각적 계층 구조
- `===` : 큰 섹션 분리
- `---` : 강한 구분
- `...` : 약한 구분
- `<<>>` : 주요 제목
- `[[]]` : 부가 정보
- `!!` : 주의사항
- `""` : 추가 설명

### 3. 색상 활용 가이드
- **파란색** (`<<>>`): 주요 제목, 핵심 정보
- **회색** (`[[]]`): 참고사항, 부가 정보
- **분홍색** (`!!`): 주의사항, 중요 안내
- **연한 파란색** (`""`): 추가 설명, 부가 혜택

### CSS 클래스명

- `.gray-group-box`: 회색 그룹박스
- `.light-pink-bg`: 연한 분홍색 배경
- `.pink-gradient-bg`: 연한 파란색 배경
- `.thick-horizontal-line`: 두꺼운 가로선
- `.dotted-separator`: 점선 구분선
- `.promotion-link`: 링크 스타일

## 📝 주의사항

1. **태그 중첩**: 태그는 중첩해서 사용할 수 없습니다.
2. **HTML 태그**: 기본 HTML 태그와 함께 사용 가능합니다.
3. **공백 처리**: 태그 내부의 공백은 보존됩니다.
4. **줄바꿈**: `\n` 문자는 자동으로 `<br>` 태그로 변환됩니다.
